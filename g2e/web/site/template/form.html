GEO2Enrichr is a web application for extracting gene sets from SOFT files and analyzing these lists for common biological functions. Choose between two methods of gene set extraction:

  1. Install the Chrome Extension to use GEO2Enrichr on the Gene Expression Omnbius website.
  2. Use the form below to upload a SOFT file. Please see our Help Page for details on file formatting.
