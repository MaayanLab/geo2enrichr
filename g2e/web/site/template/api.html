REST APIs

These REST APIs provide programmatic access to read and write data to GEO2Enrichr. You may analyze a GEO or custom SOFT file or fetch results from a previous analysis. Requests are made to the base URL:

http://amp.pharm.mssm.edu/g2e/api

Responses are in JSON.

GET api/extract/:extraction_id

Returns a JSON blob of data associated with a single user extraction, using a single extraction_id parameter. The returned data has stable links to the SOFT file; up, down, and combined gene lists; and metadata from both the experiment and analysis.

Resource URL

http://amp.pharm.mssm.edu/g2e/api/extract

Parameters

extraction_id The ID of the extraction for which to return results.

Example request

http://amp.pharm.mssm.edu/g2e/api/extract/1

POST api/extract/geo

Performs differential expression analysis on a GEO SOFT file based on GEO accession number.

Resource URL

http://amp.pharm.mssm.edu/g2e/api/extract/geo

Example response

{
    extraction_id: 1
}
