Use a browser extension

GEO2Enrichr can be used in either Chrome or Firefox. Firefox support is limited. Please follow the instructions below:

  1. Visit GEO2Enrichr on the Chrome Web Store.
  2. Click the +FREE button
  1. Download this .xpi file.
  2. Open Firefox.
  3. Click the top-right menu.
  4. Click Add-ons.
  5. Click Install Add-on From File.
  6. Navigate to the .xpi file and upload it.

See the manual for more information.
